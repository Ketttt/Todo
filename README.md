# 📝 ToDo List Application

Простое приложение для управления задачами с возможностью отмечать выполненные дела.

## 🚀 Возможности

- ✅ Добавление новых задач
- ✔️ Отметка выполненных задач
- 🔍 Поиск по задачам
- 📱 Адаптивный интерфейс
